
# EdTech Frontend Test

## Sobre

Implementação do layout proposto no Figma utilizando HTML, CSS e JavaScript vanilla.

## Tecnologias

- HTML5
- CSS3
- JavaScript Vanilla

## Como executar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador

## Estrutura do projeto

- `index.html`
- `assets/css/style.css`
- `assets/js/script.js`
- `assets/images/`

## Funcionalidades

- Hero responsivo
- Vídeo incorporado
- Slider de imagens
- Cards expansíveis
- Player de áudio customizado
- Atividade discursiva com persistência
- Atividade objetiva com validação
- FAQ com animação

## Decisões técnicas

- Uso de `sessionStorage` para persistência local
- Componentes interativos em JavaScript puro
- Organização dos assets em pastas por tipo

## Acessibilidade

- Uso de labels
- Botões semânticos
- Estados visuais de interação

## Observações

O hero foi ajustado com foco em fidelidade ao Figma e os demais blocos receberam tratamento responsivo para tablet e mobile.
